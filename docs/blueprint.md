# **App Name**: CyberLearn

## Core Features:

- Course Catalog: Display all available courses with descriptions, pricing, and instructor details.
- Course Pages: Detailed individual course pages featuring learning objectives, curriculum, instructor bio, and enrollment options.
- Enrollment: Secure online enrollment and payment processing for courses and PURCHASE THEM ALSO.
- Student Dashboard: Personalized dashboard for students to access enrolled courses, track progress, and view grades.
- About Us: Page detailing the mission, vision, and team behind Aviraj Info Tech Cyber Security.
- Intelligent Recommendation Tool: Suggest relevant courses to users using LLM analysis of course descriptions, aligning recommendations with career objectives

## Style Guidelines:

- Dark background (#121212) to create a professional and focused learning environment.
- Primary color: Vibrant blue (#3498db) for highlights and interactive elements.
- Accent color: Bright green (#2ecc71) to indicate success and call to action.
- Headline font: 'Space Grotesk', a sans-serif font, for a modern, tech-oriented aesthetic; body font: 'Inter', a sans-serif font, for readability.
- Use clean and modern icons to represent course categories and actions.
- Maintain a clean, well-organized layout to ensure ease of navigation and a distraction-free learning experience.
